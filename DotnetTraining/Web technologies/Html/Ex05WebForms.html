Using Forms of HTML 5.0

Using HTML forms

Forms are used to take inputs from the user and process it to a storage location like a centralized Database or a local Storage where the content will be saved within the browser.
Forms are usually processed in the server side. If the App is using only client side scripting, then the form does not need the attributes called action and method.
